# emulator-workshop
An emulator lets you run code written for anothor machine on your machine.
## Examples
* Run old games
* Develop a mobile app
## How do emulators work?
By implementering the behaviour of the original hardware/software with new software.

### Main loop:
* Fetch
* Decode
* Execute
